## Builder Mode

- 把 AI 能力包装成真正可用的产品，而不是只停留在模型调用层。
- 设计自动化流程，让内容生产、信息处理和交付动作尽量自驱运行。
- 更关心真实用户会不会反复使用，而不是项目看起来有多热闹。

## What I'm Building Around

- `AI-native SaaS`：把复杂任务拆成用户真正买单的产品体验
- `Automation Systems`：从采集、决策、生成到发布的完整流水线
- `Agent Workflows`：让多步骤任务具备可组合、可审核、可扩展的执行能力

## Operating Principles

- Clarity over cleverness.
- Ship the smallest useful version first.
- Turn repeated work into systems.
- Build for leverage, not just completion.
